This project was developed in 2026 after I obtained my [Python PCAP certification](https://pythoninstitute.org/pcap).  
It is also my first major project involving the use of [Prompt Engineering](https://en.wikipedia.org/wiki/Prompt_engineering).

### Why This Project

After obtaining my [Python PCAP certification](https://pythoninstitute.org/pcap), I wanted to test my skills in using AI agents and [Prompt Engineering](https://en.wikipedia.org/wiki/Prompt_engineering) through a medium-sized project.

Earning my [PCAP Python certification](https://pythoninstitute.org/pcap) was a good opportunity to supervise this project, as I strongly believe that we should always be able to **understand**, **validate**, and **improve** solutions generated by AI agents.

This project also represents valuable experience in using **Detailed Technical Specifications (DTS)** to describe and guide the development of the intended solution.

### Why JavaScript

[GitHub Pages](https://docs.github.com/en/pages) only hosts static websites (HTML/CSS/JS).  As a result, an interactive Python server-side solution is not suitable without an additional layer.\
For the final version, the original Python project was redesigned and is now implemented in **JavaScript** (client-side), with no external dependencies.

---

## Python Version Features

> **The Python version is still available and runnable using the command** `python3 main.py`.

- Offline Snake game playable directly in the terminal.
- Home menu with game start and exit options.
- Two available modes (Classic, MultiFruit).
- 5 different map sizes.
- Multiple speed levels (Slow, Normal, Fast).
- ASCII terminal rendering with visible borders.
- Real-time score display.
- Unit tests covering the core game logic.

### Pourquoi ce Projet

Après l'obtention de ma [certification Python PCAP](https://pythoninstitute.org/pcap) j'ai souhaité tester mes compétences en dans l'utilisation des agents IA et du [Prompt Engenering](https://fr.wikipedia.org/wiki/Prompt_engineering) à travers un projet de moyenne taille. 

L'obtention de ma certification Python [PCAP](https://pythoninstitute.org/pcap) a été une bonne opportunité pour superviser ce projet, étant persuadé qu'il faut toujours être en capacité de **comprendre**, **valider** et **améliorer** les solutions générées par les agents IA. 

Ce projet représente une bonne expérience de l'utilisation de [Spécifications Techniques Détaillées (STD)](https://www.akuiteo.com/glossaire-erp/specifications-techniques-detaillees/#:~:text=Les%20Sp%C3%A9cifications%20Techniques%20D%C3%A9taill%C3%A9es,de%20gestion%20et%20les%20actions.) décrivant et guidant les développement de la réalisation souhaitée.

### Pourquoi JavaScript

Pour cette version finale, le projet Python a fait l'objet d'une refonte, et est maintenant implementé en **JavaScript** (client-side), sans dependance externe.
